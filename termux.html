Termux Basic Commands (සිංහලෙන්)

ls - ෆයිල්/ෆෝල්ඩර් බලන්න
cd foldername - ෆෝල්ඩර් එකට යන්න
cd .. - එක level එකකට පිටවෙන්න
pkg install - package එකක් install කරන්න
python filename.py - Python file එකක් run කරන්න
bash filename.sh - Bash script එකක් run කරන්න
CTRL + Z - Run වෙන ප්‍රොසස් එක stop කරන්න
